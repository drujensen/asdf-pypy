# asdf-pypy

pypy plugin for https://github.com/asdf-vm/asdf/

This is a pypy plugin for asdf version manager.

# Installation

```bash
asdf plugin-add https://github.com/drujensen/asdf-pypy.git
asdf install pypy 3.8
pypy --version
```
